Darkspace Escape

Тема: Інтерактивний вебдодаток з вибором персонажа та міні-грою

Цей проєкт є курсовою роботою, що демонструє поєднання сучасних вебтехнологій для створення інтерактивного інтерфейсу та ігрової логіки у браузері.

Опис проєкту

Користувач потрапляє у світ "Darkspace Escape", де спочатку обирає свого героя з галереї доступних персонажів. Кожен герой має унікальну історію. Після вибору розпочинається міні-гра на виживання, де потрібно ухилятися від космічних загроз, збирати ресурси та протриматися якомога довше.

Основний функціонал

Вибір персонажа
Інтерактивний слайдер: Перегляд 12 унікальних персонажів.
Інфо-панель: Відображення імені, титулу та короткої історії (лору) героя.
Збереження вибору: Обраний персонаж зберігається у локальному сховищі браузера (LocalStorage) і автоматично завантажується у гру.
Міні-гра на виживання
Керування: Рух вліво та вправо за допомогою стрілок клавіатури.
Механіка:
Перешкоди: Астероїди та уламки, що падають зверху (віднімають життя).
Монети: Додають очки до рахунку.
Сердечка: Рідкісні бонуси, що відновлюють життя.
Прогресія: Швидкість гри поступово зростає.
Інтерфейс: Лічильники життя та очок у реальному часі.
Технологічний стек

HTML5: Семантична структура сторінок.
CSS3: Анімації, верстка Flexbox, стилізація інтерфейсу.
JavaScript (ES6+): Логіка інтерфейсу, слайдера та ініціалізації.
Phaser 3: Ігровий рушій для обробки фізики, колізій та рендерингу гри.
Canvas API: Використовується рушієм для відмальовування графіки.
LocalStorage: Збереження стану між сторінками.

Інструкція з розгортання та запуску

Для коректного функціонування розробленого програмного продукту на пристрої користувача має бути встановлений веб-браузер останньої стабільної версії з підтримкою стандартів HTML5 та JavaScript ES6 (наприклад, Google Chrome, Mozilla Firefox, Microsoft Edge). Для першого запуску гри необхідне активне підключення до мережі Інтернет для завантаження бібліотеки Phaser 3.

Спосіб 1: Через Git 1.Відкрийте термінал або командний рядок. 2.Виконайте команду клонування репозиторію: git clone https://github.com/aleksakosogova/coursework.git

Спосіб 2: Завантаження ZIP-архіву 1.Перейдіть на сторінку репозиторію GitHub. 2.Натисніть зелену кнопку Code. 3.Оберіть опцію Download ZIP. 4.Розпакуйте завантажений архів у зручну папку на комп'ютері.

Оскільки політики безпеки сучасних браузерів (CORS) можуть блокувати завантаження локальних зображень та скриптів при безпосередньому відкритті HTML-файлу, для коректного запуску рекомендується використовувати локальний вебсервер.

Покрокова інструкція запуску через Visual Studio Code: 1.Завантажте папку з проєктом на свій комп'ютер. 2.Запустіть редактор коду Visual Studio Code. 3.У меню оберіть пункт File (Файл) -> Open Folder (Відкрити папку) та вкажіть шлях до папки з проєктом. 4.Встановіть розширення Live Server через вбудований магазин розширень редактора (якщо воно не було встановлено раніше). 5.У провіднику файлів редактора (зліва) розгорніть папку character-game-generator та знайдіть файл index.html. 6.Натисніть правою кнопкою миші на файл index.html. 7.У контекстному меню оберіть пункт Open with Live Server. 8.Вебдодаток автоматично відкриється у вашому браузері за адресою локального хоста (зазвичай https://www.google.com/search?q=http://127.0.0.1:5500).

Інструкція користувача

Головне меню: Після завантаження натисніть "Обрати героя", щоб перейти до галереї. Вибір персонажа: Використовуйте стрілки на екрані для перегляду героїв. Ознайомтеся з описом та натисніть "Обрати персонажа" для підтвердження вибору, а потім "Почати гру". Ігровий процес: Керування: Використовуйте клавіші стрілок Вліво (←) та Вправо (→) на клавіатурі для переміщення персонажа. Мета: Уникайте зіткнень з астероїдами та камінням. Бонуси: Збирайте монети (жовті круги) для очок та серця (червоні) для відновлення життя. Завершення: Гра закінчується, коли кількість життів дорівнює нулю. Для перезапуску натисніть кнопку повернення в меню.
