Тема: Інтерактивний генератор анімованих персонажів з міні-грою
Опис проекту: Вебдодаток, що дозволяє користувачу створити унікального анімованого персонажа, обираючи та комбінюючи різні елементи (зовнішність, одяг, аксесуари та інше). Створений персонаж потім інтегрується у просту гру на виживання, де користувач керує ним та ухиляється від об'єктів, що падають.

Основні функції
Генератор персонажів
- Вибір кольору та форми тіла
- Налаштування голови та риси обличчя
- Вибір одягу та аксесуарів
- Перегляд персонажа в реальному часі
- Збереження конфігурації персонажа
Міні-гра на виживання
- Керування персонажем за допомогою стрілок клавіатури
- Падаючі об'єкти, які необхідно уникати
- Система очків
- Анімація руху персонажа

Технологічний стек
- Платформа: Веб-додаток (Desktop/Responsive)
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Графіка та анімація: Canvas API
- Ігрова логіка: Нативний JavaScript
- Збереження даних: LocalStorage для персонажів користувача
- Адаптвність: Медіа-запити для різних розмірів екранів

Реалізація
Генератор персонажа
Користувач обирає з набору параметрів:
- Колір тіла (3-4 варіанти)
- Колір голови (3-4 варіанти)
- Вид очей (2-3 варіанти)
- Колір одягу (3-4 варіанта)
Персонаж відображається як просте креслення на Canvas та оновлюється при кожному виборі.
Гра на виживання
- Персонаж розташований внизу екрану
- Об'єкти падають з верхньої частини
- Гравець керує горизонтальним рухом персонажа за допомогою стрілок
- При зіткненні з об'єктом — втрата очка
- Гра закінчується при досягненні певної кількості втрач
